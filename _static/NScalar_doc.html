NScalar reference¶

This is a special kind of vector, basically it rapresent a vector of size one, aka one element only, having this class lets you target attributes which are not compound

class nVec.NScalar(attribute_name, base_name='Vector')

Bases: nVec.NBaseVec

This class represents a single float number used for vector operation, in pratice it reflects a VEC1 (vec of length one) that is needed in some vector operation, like scalar value operation
